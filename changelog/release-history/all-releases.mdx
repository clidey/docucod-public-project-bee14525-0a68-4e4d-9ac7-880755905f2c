# All Releases & Changelog

Welcome to the centralized timeline of Linkwarden's journey through continuous innovation and improvement. This page offers a well-organized, chronological list of all product releases, making it effortless for users to track changes, discover new features, understand fixes, and quickly access detailed release notes.

Whether you're a long-time user or just starting, this overview is designed to help you stay informed and make the most out of Linkwarden's evolving capabilities.

---

## Navigating the Changelog

The changelog entries are presented in reverse chronological order, with the most recent releases at the top. Each release entry contains:

- **Version Number and Release Date**: Clearly indicates when the release became available.
- **Summary of Changes**: A concise, user-focused overview of what’s new, improved, or fixed.
- **Links to Detailed Release Notes**: For in-depth information, troubleshooting, and upgrade advice.

This structure allows you to grasp the impact quickly and decide if you want to dive deeper into specific updates.

## How to Use This Page

### Tracking New Features and Improvements

Look here first whenever you want to find what new capabilities have been introduced or how existing features have been enhanced. Each summary highlights benefits and practical value, such as improved bookmarking workflows, enhanced archival options, or new collaboration tools.

## Recent Releases

### Version 1.36.0 – 2024-06-15
- Introduced AI-powered tag suggestions to help you categorize bookmarks faster and smarter.
- Enhanced archival options now support capturing full-page monolith and improved screenshot fidelity.
- Fixed issue where bulk editing tags could inadvertently overwrite existing tag assignments.
- Improved synchronization speed for collaborative collections.

[Detailed Release Notes](https://github.com/linkwarden/linkwarden/releases/tag/v1.36.0)

### Version 1.35.2 – 2024-05-10
- Patched a security vulnerability affecting access tokens expiration handling.
- Fixed UI glitch in the mobile app causing occasional crashes during link search.
- Improved API error messages for better integration diagnostics.

[Detailed Release Notes](https://github.com/linkwarden/linkwarden/releases/tag/v1.35.2)

### Version 1.35.0 – 2024-04-22
- Added public sharing settings per collection for more granular privacy control.
- Optimized the link import process with support for Wallabag and Pocket formats.
- Introduced a new dashboard section to display pinned links prominently.
- Fixed synchronization delays occurring under high network latency.

[Detailed Release Notes](https://github.com/linkwarden/linkwarden/releases/tag/v1.35.0)
